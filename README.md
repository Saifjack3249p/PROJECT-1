# 🐍 Snake Water Gun Game (Python)

A simple command-line game built using Python where you play against the computer.

---

## 🎮 Game Description

Snake Water Gun is similar to Rock Paper Scissors. The player chooses one option, and the computer randomly selects another. The winner is decided based on predefined rules.

---

## 📌 Rules

* 🐍 Snake drinks Water → Snake wins
* 💧 Water destroys Gun → Water wins
* 🔫 Gun kills Snake → Gun wins
* Same choices → Draw

---

## 🚀 Features

* Play against computer
* Random computer choices using Python
* Score tracking system
* Continuous gameplay using loop
* Exit anytime using `exit`

---

## 🛠️ Technologies Used

* Python (Core concepts)
* Random module
* Conditional logic
* Loops

---

## ▶️ How to Run

1. Clone this repository:

```
git clone https://github.com/your-username/snake-water-gun.git
```

2. Navigate to the folder:

```
cd snake-water-gun
```

3. Run the program:

```
python game.py
```
